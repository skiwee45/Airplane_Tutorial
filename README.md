# Airplane Tutorial
Physics Based Airplane Controller (Unity)

Based on [__Aircraft-Physics__](https://github.com/gasgiant/Aircraft-Physics)

## Modifications Made:
- Controllable thrust
- Added flaps to wings (used for takeoff and landing)
- Added ground breaks
